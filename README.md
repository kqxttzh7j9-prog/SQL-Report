# SQL-Report

Fixed Income Portfolio Analysis — SQLite Case
A Python-based portfolio analysis tool built around an SQLite database, designed to simulate and evaluate a fixed income portfolio. The project covers NAV calculations, PnL attribution, and key risk metrics including Sharpe ratio and maximum drawdown. Output is generated as an automated PDF report.
Built as a practical case demonstrating end-to-end data handling: from raw bond data stored in a relational database to structured performance reporting.
Stack: Python, SQLite, pandas, reportlab
